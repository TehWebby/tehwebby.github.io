What is it

- -

Keep Tracking

Keep a record of all your favourite films in organised lists. With Risuto's tracking features you can keep an eye on what you've watched and when you've watched it.

Up to date

With the help of themoviedatabase.org, Risuto allows you to keep track of the latest movies available whilst providing movie statistics on the move.

Cross-Platform Application

Risuto was developed in the latest Cross-Platform technologies which means it can be accessed on iOS and Android devices (and maybe even Windows devices).

Development Timeline

Where it all began

  • October 2015

    Team Formed

    Luke pitches the idea in the Mobile Application Design Lesson and the team is formed.

  • October 2015

    Architecture Chosen

    After much deliberation, the team finally decided on the architecture and frameworks that would be used in the application.

  • November 2015

    Node Server Went Live

    After a lot of blood, sweat and tears, Shaun & Josh finally succeeded in getting the Node server live to handle user requests.

  • January 2016

    Application Deployed to Device

    The team finally gets the application onto both an iOS and Android device. Absolute happiness followed by absolute agony when the team realise the number of bugs that can be found on these devices.

  • March 2016

    Production-Ready Application

    Risuto is finally ready to be shown to the world! Well, maybe not quite the world, but you guys will do for now.
