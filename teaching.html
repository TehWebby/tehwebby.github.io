Number

Unlike many other languages the number data type includes integers and floats.
Example: 1, -2 and 3.43 are classed as a number data type.
This is different from many languages that have multiple data types to support different numbers.
JavaScript is good and bad in this way. It allows developers to worry less about small trivial but gives less control over the code.

typeof num; // number

var num = 3.43;
typeof num; // number
